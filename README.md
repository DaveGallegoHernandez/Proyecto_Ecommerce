# Ecommerce

First install node js then:

```bash
 $ npm i

 // Or

 $ npm install

```

Run the project with:

```bash
 $ npm run dev
```

This works on [http://localhost:3000](http://localhost:3000)

## Features

- [ ] Translations (en, es)
- [ ] Auth
    - [ ] Roles ( admin, user, moderator)
- [ ] Products and pagination
- [ ] News letter
